# Publishing Checklist

Use this checklist before publishing any new page or article.

- Confirm the content is evergreen or freshly verified.
- Avoid fake rankings, fake reviews, and fake download counts.
- Avoid claims that make NewGames.ai sound like an official ranking authority.
- Verify any third-party game facts before publishing.
- Use simple SVG placeholder images unless a future image policy allows more.
- Confirm title, description, categories, tags, cover, coverAlt, and draft status.
- Run `./tools/check_site.sh`.
- Review the generated page locally.
