# Image Policy

NewGames.ai uses a cautious image strategy to keep the site lightweight, safe, and easy to maintain.

## MVP Stage

- Use static SVG placeholders stored in the GitHub repo.
- Keep images abstract and lightweight.
- Use existing category and default OG images for starter posts.
- Do not use third-party screenshots, logos, characters, or store UI.

## Early Stage

- Reuse category images across posts.
- Prefer the default OG image when a post does not need a unique visual.
- Keep visuals consistent with the NewGames.ai discovery style.
- Avoid adding image infrastructure before the content volume justifies it.

## Mid Stage

- Selected AI-generated abstract images may be used for important evergreen posts.
- AI image prompts must avoid real games, characters, logos, store interfaces, and screenshot-like scenes.
- Human review is required before adding generated images to the site.
- Images should support evergreen content rather than chase short-term trends.

## Long Term

- Cloudflare R2 may be considered for media storage using `media.newgames.ai`.
- Cloudflare Images should be avoided until image volume, transformation needs, or operational requirements justify it.
- Any future media pipeline should include review, naming, alt text, and licensing checks.

## Prohibited Image Use

- No third-party screenshots unless properly licensed.
- No AI images that imitate real games, characters, logos, store UI, or screenshots.
- No trademark imitation.
- No copied promotional art.
- No images that imply an official partnership when none exists.
