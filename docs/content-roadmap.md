# Content Roadmap

## Launch Topics

- Beginner-friendly game discovery guides.
- Mobile games for quick sessions.
- Web games and browser games for short breaks.
- Puzzle games for casual players.
- Indie game discovery habits.
- Casual game selection tips.

## Future Topic Clusters

- New game discovery checklists.
- Platform-based discovery pages.
- Genre-based evergreen guides.
- BornstarSoft and Gempia network updates.
- Seasonal roundups that are manually reviewed before publishing.

## Editorial Guardrails

- Keep recommendations practical and transparent.
- Use verified details for named games.
- Keep general advice evergreen when facts are not verified.
- Prefer helpful curation over hype.
